About the project

01

Introducing

OVERVIEW
‘미니언즈 제주 특별전’은 제주도에 위치한 한화 아쿠아 플라넷에서 영화 ‘슈퍼배드’의 귀요미 캐릭터 미니언즈를 주제로
열리는 특별전입니다. 애니메이션 제작 과정을 담은 아트웍을 비롯해 실물 모형, 인터렉티브 체험물, 특별 제작 굿즈, 포토존을 통해
추억을 남길 수 있는 ‘미니언즈 제주 특별전’과 함께 진행되는 카드사 할인 이벤트를 소개하는 프로모션 페이지를 제작하였습니다.

STYLE

STYLE GUIDE

캐릭터의 만화적인 요소와 매칭될 수 있도록 주요 색상에 채도가 높은 컬러를 사용했습니다. 채도가 높은 배경에서도 특별전의
주제와 내용을 이해하는데 용이하도록 굵은 산세리프체를 사용하여 가독성을 높이고자 하였습니다. 특별전의 구성을 설명하는
부분에서는 미니언즈의 귀여운 이미지와 매칭되도록 소제목 타이틀을 둥근 타이포로 표현하였습니다.

02

Main page

Detail

MAIN PAGE

미니언즈 캐릭터가 주제인 전시 기획에 맞춰 미니언즈 캐릭터 이미지를 활용하여 각각의 컨텐츠의 내용에 어울리도록
콜라주하여 작업했습니다. 많은 정보가 담겨 있기 때문에 느낄 수 있는 피로감과 컨텐츠 사이의 단절감을 줄이기 위해
상단에 표현한 공간감을 하단 컨텐츠에서도 활용하여 배경 컬러를 하늘에서 바닷속으로 들어가는 느낌으로 표현했습니다.
전체적인 구성은 ( 상단 표지 - 전시의 개괄적인 소개 - 전시가 열리는 장소 - 전시 내용의 설명 - 이벤트 페이지 ) 순 입니다.
TOP PAGE
해변을 즐기고있는 캐릭터들의 이미지와
제주 관광 명소인 성산일출봉 이미지를
합성하여 상단을 구성, 타이틀의 컬러와
디테일을 통해 제주도의 여행지 이미지와
공간감이 돋보일 수 있도록 제작했습니다.
INTRO PAGE
간략한 소개말과 영상으로 구성된
콘텐츠 입니다. 상단 표지 타이틀과
같은 폰트와 디테일로 중제목을 표현
하여 통일감을 유지하였고 소개 영상
을 평면적인 느낌보다는 입체감있게
표현하고 싶었습니다.
LOCATION PAGE
특별전이 열리는 테마파크의 안내 및
소개부분입니다. 아쿠아리움의 관람
포인트들을 다양하게 보여주고 홍보가
될 수 있도록 슬라이드 형식을 활용하여
구성하였습니다.
VIEW POINT PAGE
특별전의 관람 구성의 순서에 맞춰 자세
한 전시 컨텐츠를 설명하고 있는 부분이며,
각각의 이미지에 포토존 느낌의 디테일을
추가하여 고객의 니즈를 자극하고 충족
시킬 수 있음을 어필하고자 하였습니다.
EVENT PAGE
특정 카드사 회원에게만 제공되는 할인
이벤트의 안내와 미니언즈 카드의 홍보
및 신규 발급을 유도하는 컨텐츠입니다.
미니언즈 팬의 니즈를 자극할 수 있도록
실물 카드를 상단에 크게 배치하였으며
카드사 홈페이지로 바로 이동하여 카드
발급을 신청할 수 있도록 링크 배너도
함께 디자인 하였습니다.
